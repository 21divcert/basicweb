# 우주게임_2021108240_강다연

### 게임소개
- 하늘에서 랜덤으로 enemy 비행선이 내려오는데, 비행선을 레이저를 쏘며 무찌르는 게임입니다.
- 레이저를 쏠 때마다 레이저의 색이 바뀝니다.

### 게임방법
- 방향키(상,하,좌,우)를 이용하여 hero 비행선을 움직일 수 있습니다.
- hero 비행선의 생명은 3개로 enemy 비행선과 3번 충돌 시, 게임이 종료됩니다.
- 모든 적의 비행선을 무찌르면 성공합니다.
